## 📌 Project Overview

This project analyzes an **Online Bookstore dataset using SQL** to extract meaningful insights about **book sales, customer behavior, and inventory management**.

The goal of this project is to practice **SQL querying skills** by solving real-world analytical questions using **basic and advanced SQL queries**.

---

## 🎯 Project Objectives

- Analyze bookstore sales data
- Understand customer purchasing patterns
- Identify best-selling books and genres
- Evaluate inventory stock levels
- Generate business insights using SQL queries

---

## 🗂 Dataset Structure

The project uses the following tables:

| Table Name | Description |
|------------|-------------|
| Books | Contains book details such as title, author, genre, price, and stock |
| Customers | Stores customer information including name, city, and country |
| Orders | Contains order details such as order date and total amount |
| Order_Details | Connects orders with books and quantities purchased |

---

## 🧠 SQL Assignment Questions

### 🔹 Basic Queries

1. Retrieve all books in the **"Fiction" genre**  
2. Find books published **after 1950**  
3. List all customers from **Canada**  
4. Show orders placed in **November 2023**  
5. Retrieve the **total stock of books available**  
6. Find the **most expensive book**  
7. Show customers who ordered **more than 1 quantity**  
8. Retrieve orders where **total amount exceeds $20**  
9. List all **genres available** in the books table  
10. Find the **book with the lowest stock**  
11. Calculate the **total revenue generated**

---

### 🔹 Advanced Queries

1. Total **books sold for each genre**  
2. Average **price of books in Fantasy genre**  
3. Customers who placed **at least 2 orders**  
4. **Most frequently ordered book**  
5. Top **3 most expensive Fantasy books**  
6. Total **books sold by each author**  
7. Cities where customers **spent over $30**  
8. Customer who **spent the most**  
9. Remaining **stock after fulfilling all orders**
